# Git-OSINT

Git-OSINT is a web-based tool designed for Open Source Intelligence (OSINT) gathering on Git repositories. It provides a user-friendly interface to analyze and investigate git repositories for valuable information.

## Features

*   **Repository Analysis:** Dive deep into repository data.
*   **Contributor Insights:** Identify top contributors and their activity patterns.
*   **Commit History Search:** Search through commit messages and code changes for specific keywords or patterns.
*   **Sensitive Information Detection:** Scan for potential leaks of sensitive data like API keys, passwords, and private keys.
*   **Visualizations:** (Planned) Visualize repository activity, contributor networks, and code frequency.

## How to Use

This is a static web application. No installation is required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AnkurMacwan/git-osint.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd git-osint
    ```
3.  **Open the application:**
    Open the `index.html` file in your web browser.
